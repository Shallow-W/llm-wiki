# Deshraj Yadav

## 概述

Deshraj Yadav 是 Mem0（后更名为 Letta）的联合创始人和 CEO，专注于 AI Agent 记忆系统和多模态 AI 研究。他领导团队开发了 Mem0 和 Mem0g 记忆架构，在 LOCOMO 长对话记忆基准上实现了全面领先。

## 关键事实

- **职位**：Mem0 / Letta 联合创始人、CEO
- **研究方向**：AI Agent 记忆、长期对话一致性、多模态 AI
- **前经历**：Facebook AI Research（FAIR），视觉与语言理解方向
- **代表论文**：Mem0: Building Production-ready AI Agents with Scalable Long-term Memory (2025)

## 主要贡献

1. **Mem0 记忆架构**：提出增量式文本记忆提取与更新系统
2. **Mem0g 图记忆**：将图结构引入 Agent 记忆，增强时序和关系推理
3. **LOCOMO 系统评估**：建立记忆系统的全面评估方法论
4. **生产级部署**：实现 91% 延迟降低和 90%+ token 成本节省
