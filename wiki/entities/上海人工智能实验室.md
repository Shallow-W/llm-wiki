# 上海人工智能实验室

## 概述
上海市设立的人工智能研究机构，是 EvolveR、O2-Searcher 等多项 LLM Agent 研究的主要产出机构。

## 关键事实
- 成立时间：上海
- 核心领域：人工智能基础研究、LLM、自主 Agent、计算机视觉、自动驾驶
- 关联机构：上海创新研究院、上海交通大学（合作关系）
- 知名成果：EvolveR（经验驱动自演化 Agent）、O2-Searcher（开放域搜索 Agent）
