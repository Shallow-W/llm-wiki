# Chroma

## 概述

Chroma 是一家专注于向量数据库和检索基础设施的 AI 公司，为大型语言模型（LLM）应用提供嵌入存储和检索能力。公司以其开源向量数据库 Chroma DB 而闻名，同时也积极从事 LLM 长上下文和记忆机制的研究。

## 关键事实

- **核心产品**：Chroma DB — 开源向量数据库，支持嵌入存储、相似度搜索和元数据过滤
- **研究领域**：LLM 长上下文评估、上下文腐烂（Context Rot）、检索增强生成（RAG）
- **知名报告**：[[source-context-rot|Context Rot: How Increasing Input Tokens Impacts LLM Performance]]（2025 年 7 月）
- **官网**：https://trychroma.com
