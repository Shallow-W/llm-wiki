# Bharadwaj Veeravalli

## 概述

新加坡国立大学电气与计算机工程系终身教授，IEEE 高级会员，可分负载理论（DLT）领域的早期研究者之一。

## 关键事实

- 学位：物理学学士（1987，马杜赖卡马拉杰大学）、电子通信工程硕士（1991，印度科学院）、博士（1994，印度科学院），均获金质奖章
- 职位：新加坡国立大学电气与计算机工程系终身教授
- 研究方向：云计算/网格/集群计算、并行分布式系统调度、多媒体计算、可分负载理论（DLT）
- 编辑职务：IEEE Transactions on Cloud Computing 副编辑
