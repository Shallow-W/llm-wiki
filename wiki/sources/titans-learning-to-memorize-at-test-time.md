# Titans: Learning to Memorize at Test Time

## 一句话总结

提出一种深度神经长期记忆模块，通过在线梯度下降（带动量和权重衰减）在测试时持续学习记忆历史信息，并基于此构建 Titans 架构家族（MAC/MAG/MAL 三种变体），在语言建模、常识推理、长上下文检索、时间序列预测和基因组学任务上全面超越 Transformer 和现代线性循环模型。

## 研究背景与动机

### 现有架构的记忆困境

论文从"记忆视角"重新审视了现有序列模型：

| 架构 | 记忆形式 | 更新方式 | 核心局限 |
|------|----------|----------|----------|
| **RNN/LSTM** | 向量值隐状态 $h_t \in \mathbb{R}^d$ | 压缩式更新 $f(h_{t-1}, x_t)$ | 信息压缩损失严重，长期依赖困难 |
| **Transformer** | 增长的 KV 矩阵（上下文窗口内） | 追加式（无压缩） | 二次复杂度，上下文长度受限 |
| **线性 Transformer** | 固定大小矩阵值记忆 $M_t$ | 加性压缩 $M_t = M_{t-1} + K_t^\top V_t$ | 记忆溢出（additive overflow） |
| **现代线性 RNN (Mamba等)** | 向量/矩阵值记忆 + 遗忘门 | 数据依赖的遗忘 + 压缩 | 线性记忆，表达能力受限；缺乏深度记忆 |
| **DeltaNet** | 矩阵值记忆 | Delta-rule 更新 | 仅有瞬时惊讶，缺乏动量；无遗忘机制 |

### 五个核心问题 (Q1-Q5)

论文提出了五个关键问题来指导设计：

1. **Q1**：什么是好的记忆结构？
2. **Q2**：什么是合适的记忆更新机制？
3. **Q3**：什么是好的记忆检索过程？
4. **Q4**：如何设计一个有效整合不同记忆模块的架构？
5. **Q5**：是否需要深度记忆模块来有效存储长期历史？

### 核心洞察

> **注意力（有限上下文、精确依赖建模）扮演"短期记忆"角色；神经记忆（持续学习、参数化存储）扮演"长期记忆"角色。**

人类记忆系统由多个独立运作的子系统组成（短期、工作、长期记忆），Titans 的设计正是受此启发。

## 核心方法

### 3.1 神经长期记忆模块 (Neural Long-term Memory)

#### 惊讶度量 (Surprise Metric)

核心直觉：**违反预期的事件更令人难忘**。论文用梯度作为惊讶度量：

$$M_t = M_{t-1} - \theta_t \nabla \ell(M_{t-1}; x_t)$$

其中 $\nabla \ell(M_{t-1}; x_t)$ 是损失函数对记忆的梯度，梯度越大表示输入越"出乎意料"。

但纯梯度惊讶有个问题：在连续多个惊讶 token 后，梯度可能变得极小（陷入平坦区域），导致后续重要信息被遗漏。为此，论文将惊讶分解为两部分：

$$M_t = M_{t-1} + S_t$$
$$S_t = \underbrace{\eta_t S_{t-1}}_{\text{Past Surprise}} - \underbrace{\theta_t \nabla \ell(M_{t-1}; x_t)}_{\text{Momentary Surprise}}$$

- **Past Surprise**（过去惊讶）：$\eta_t$ 是数据依赖的惊讶衰减，控制历史惊讶如何影响当前
- **Momentary Surprise**（瞬时惊讶）：$\theta_t$ 控制当前梯度惊讶的权重

这本质上就是**带动量的梯度下降**，其中 $S_t$ 是动量项，作为"跨时间的惊讶记忆"。

#### 关联记忆目标函数

记忆模块学习键值关联。给定输入 $x_t$，投影为 key 和 value：

$$k_t = x_t W_K, \quad v_t = x_t W_V$$

损失函数为：

$$\ell(M_{t-1}; x_t) = \frac{1}{2} \|M_{t-1}(k_t) - v_t\|_2^2$$

在元学习框架下，$W_K, W_V$ 是超参数（外循环优化），$M$ 的权重在内循环中通过上述损失优化。

#### 遗忘机制 (Forgetting Mechanism)

处理百万级 token 序列时，必须管理记忆容量。论文引入自适应遗忘：

$$M_t = (1 - \alpha_t) M_{t-1} + S_t$$
$$S_t = \eta_t S_{t-1} - \theta_t \nabla \ell(M_{t-1}; x_t)$$

其中 $\alpha_t \in [0, 1]$ 是门控机制：
- $\alpha_t \to 0$：保留旧记忆，仅更新
- $\alpha_t \to 1$：清空整个记忆

论文证明这一权重衰减机制等价于现代 RNN 中的门控遗忘机制（如 Mamba）。

#### 深度记忆架构

论文使用 $L_M \geq 1$ 层的 MLP 作为记忆模块。关键论点：

> 向量/矩阵值记忆等价于在线线性回归，假设历史数据的底层依赖是线性的。而深度记忆模块（$L_M \geq 2$）严格比线性模型更具表达能力（Hornik et al., 1989）。

实验证实：更深的记忆在更长序列上表现更好（图7）。

#### 记忆检索

检索时**不更新权重**（纯前向传播）：

$$q_t = x_t W_Q, \quad y_t = M^*(q_t)$$

其中 $M^*$ 表示前向传播但不更新参数。

### 3.2 并行化训练算法

神经记忆的训练等价于用带动量+权重衰减的梯度下降优化关联记忆损失。直接实现需要 $O(N)$ 顺序计算。论文通过以下技巧实现并行化：

**分块 mini-batch 梯度下降**：将序列分为大小为 $b$ 的块，利用矩阵乘法重写更新：

$$M_t = \beta_t M_0 - \sum_{i=1}^{t} \frac{\theta_i \beta_t}{\beta_i} \nabla \ell(M_{t'}; x_i)$$

其中 $\beta_i = \prod_{j=1}^{i}(1 - \alpha_j)$。对于线性记忆 $M_t = W_t$：

$$\sum_{i=1}^{b} \frac{\theta_i \beta_b}{\beta_i} \nabla \ell(W_0; x_i) = \Theta_b B_b (W_0 X - X) X^\top$$

**动量项的并行化**：动量递推 $S_t = \eta_t S_{t-1} - \theta_t u_t$ 是线性递推，可用**并行关联扫描 (parallel associative scan)** 在 $O(\log N)$ 深度内完成。

**块级参数简化**：将 $\alpha_t, \theta_t, \eta_t$ 设为块内常数，可进一步加速（利用全局卷积计算 LTI 系统）。

### 3.3 持久记忆 (Persistent Memory)

除上下文依赖的长期记忆外，Titans 还引入**数据独立**的可学习参数 $P = [p_1, p_2, ..., p_{N_p}]$，拼接在序列开头：

$$x_{\text{new}} = [p_1, p_2, ..., p_{N_p}] \| x$$

三个动机：
1. **记忆视角**：存储任务知识（如何完成任务的抽象）
2. **前馈网络视角**：类似 FFN 中数据独立的注意力权重（Sukhbaatar et al., 2019）
3. **技术视角**：缓解因果注意力对初始 token 的隐性偏置

### 4. Titans 架构家族：三种记忆整合方式

基于上述模块，论文提出三种变体：

#### 4.1 Memory as a Context (MAC)

将长期记忆作为当前上下文的"额外信息源"。序列分块后，当前段 $S^{(t)}$ 查询历史记忆：

$$h_t = M_{t-1}^*(q_t), \quad q_t = S^{(t)} W_Q$$
$$\tilde{S}^{(t)} = [p_1, ..., p_{N_p}] \| h_t \| S^{(t)}$$
$$y_t = \text{Attn}(\tilde{S}^{(t)})$$

然后更新记忆：$M_t = M_{t-1}(y_t)$，最终输出：$o_t = y_t \otimes M_t^*(y_t)$

**优势**：
- 注意力能自主决定是否需要长期记忆信息
- 注意力帮助记忆筛选有用信息，避免记忆溢出
- 测试时：持久记忆固定、注意力做上下文学习、长期记忆持续学习

#### 4.2 Memory as a Gate (MAG)

滑动窗口注意力（SWA）作为短期记忆，神经记忆作为长期记忆，通过门控融合：

$$\tilde{x} = [p_1, ..., p_{N_p}] \| x$$
$$y = \text{SW-Attn}^*(\tilde{x})$$
$$o = y \otimes M(\tilde{x})$$

其中 $\otimes$ 是可学习的非线性门控（归一化 + 非线性激活）。

#### 4.3 Memory as a Layer (MAL)

更传统的堆叠方式：记忆层在前，注意力层在后：

$$\tilde{x} = [p_1, ..., p_{N_p}] \| x$$
$$y = M(\tilde{x})$$
$$o = \text{SW-Attn}(y)$$

**局限**：模型能力受限于各层独立处理，无法利用注意力和神经记忆的互补数据处理。

#### 理论结果

**Theorem 4.1**：与 Transformer、对角线性循环模型和 DeltaNet（均限于 TC$^0$）不同，Titans 能解决 TC$^0$ 之外的问题，意味着 Titans 在状态跟踪任务中理论上比 Transformer 和大多数现代线性循环模型更具表达能力。

## 优点与局限

### 优点

1. **测试时学习**：记忆权重在推理时持续更新，实现真正的"终身学习"
2. **深度记忆**：$L_M \geq 2$ 的 MLP 记忆比线性/矩阵记忆更具表达能力
3. **动量式惊讶**：不仅考虑瞬时惊讶，还累积历史惊讶流，避免信息遗漏
4. **自适应遗忘**：数据依赖的权重衰减有效管理有限记忆容量
5. **并行训练**：通过分块 matmul 和关联扫描实现高效并行化
6. **理论保证**：超越 TC$^0$ 的表达力，优于 Transformer 和线性循环模型
7. **多任务泛化**：在语言、时间序列、基因组学等多领域有效
8. **超长上下文**：可扩展到 2M+ 上下文窗口，NIAH 准确率显著优于基线

### 局限与开放问题

1. **模型规模有限**：当前实验仅到 760M 参数，更大规模（数B参数）的效果待验证
2. **训练数据量**：最大模型仅训练 30B tokens，与 Llama 等（数T tokens）差距大
3. **记忆深度权衡**：更深的记忆效果更好但训练更慢，最优深度如何选择？
4. **块大小选择**：并行化依赖分块大小 $b$，影响效率和效果的权衡
5. **与预训练 LLM 的结合**：论文从头训练小模型，如何与已有大模型结合？
6. **多模态扩展**：当前仅在文本、时间序列、DNA 上验证，视觉/音频等模态待探索
7. **计算开销**：虽然训练可并行，但深度记忆的推理仍比纯注意力慢

## 与 δ-mem 的关联

### 联系

| 维度 | Titans | δ-mem |
|------|--------|-------|
| **核心思想** | 测试时学习记忆 | 在线关联记忆 |
| **更新规则** | 带动量的梯度下降 + 权重衰减 | 门控 delta-rule |
| **惊讶机制** | 梯度作为惊讶度量 | 预测误差 $(v_t - S_{t-1}k_t)$ 作为更新信号 |
| **遗忘机制** | 数据依赖的权重衰减 $\alpha_t$ | 数据依赖的保留门 $\lambda_t = 1 - \beta_t$ |
| **目标** | 将历史压缩到记忆参数中 | 将历史压缩到固定大小状态矩阵 |
| **与注意力关系** | 注意力 = 短期记忆，神经记忆 = 长期记忆 | 记忆状态直接低秩修正注意力计算 |

**本质联系**：两者都试图解决"如何在有限容量下有效记忆历史"的问题，且都使用了 delta-rule 的变体（预测误差驱动更新）和门控遗忘机制。

### 区别

| 维度 | Titans | δ-mem |
|------|--------|-------|
| **架构定位** | 替换/重新定义骨干架构 | 附加在冻结的 Transformer 骨干上 |
| **记忆结构** | 深度 MLP ($L_M \geq 1$) | 单层矩阵状态 $S \in \mathbb{R}^{r \times r}$ ($r=8$) |
| **参数规模** | 完整模型参数（数百M） | 仅 4.87M 可训练参数（0.12%） |
| **训练方式** | 端到端训练整个架构 | 冻结骨干，仅训练记忆模块 |
| **与注意力的关系** | 三种整合方式（MAC/MAG/MAL） | 记忆读出直接修正注意力 query 和 output |
| **动量机制** | 显式动量项 $S_t$ 累积历史惊讶 | 无显式动量，仅单步 delta-rule |
| **应用场景** | 从头训练序列模型 | 增强已有 LLM 的记忆能力 |
| **持久记忆** | 显式引入可学习持久记忆参数 | 无此设计 |

**关键区别**：
- Titans 是**架构级创新**——重新定义了序列模型的基本构建块
- δ-mem 是**插件级创新**——在不改变骨干的前提下增强记忆能力
- Titans 的**深度记忆**和**动量机制**是 δ-mem 所没有的，但也因此更"重"
- δ-mem 的**极致轻量**（8×8 矩阵）使其更容易部署到现有模型，但容量上限也更明显

### 互补性

两者可以视为同一思想光谱上的不同选择：
- **Titans** 适合从头设计新架构、追求极致长上下文性能的场景
- **δ-mem** 适合已有模型增强、资源受限、快速部署的场景
- 理论上，Titans 的神经记忆模块可以视为 δ-mem 的"深度版"，而 δ-mem 可以视为 Titans 的"轻量部署版"

## 讨论的概念

- [[Titans]] — 论文提出的架构家族，含 MAC/MAG/MAL 三种变体
- [[神经长期记忆]] — 深度 MLP 记忆模块，测试时通过梯度下降学习
- [[delta-rule-learning]] — 记忆更新的核心规则（预测误差驱动）
- [[测试时学习]] — 模型在推理时持续更新参数
- [[注意力机制]] — 被重新诠释为"短期记忆"
- [[线性循环模型]] — 现代高效序列模型（Mamba、DeltaNet 等）
- [[δ-mem（在线关联记忆）]] — 同为测试时记忆，但更轻量、附加式
- [[持久记忆]] — 数据独立的可学习任务知识参数
- [[惊讶度量]] — 用梯度衡量输入的"意外程度"

## 推荐参考文献

### 强烈推荐

| # | 论文 | 推荐理由 |
|---|------|----------|
| 1 | **δ-mem: Efficient Online Memory for Large Language Models** (Lei et al., 2026) | Titans 的轻量级"表亲"——同样的 delta-rule + 遗忘思想，但用 8×8 矩阵附加在冻结骨干上。理解 Titans 后读 δ-mem，可看到同一理念的不同工程选择。 |
| 2 | **TTT: Test-Time Training Layers** (Sun et al., 2024) | 与 Titans 最接近的基线。同样使用梯度下降在测试时更新记忆，但缺乏动量和遗忘机制。论文附录 C 详细对比了两者差异。 |
| 3 | **Gated DeltaNet** (Yang et al., 2024) | Titans 的直接前身之一。论文证明 LMM 是 Gated DeltaNet 的广义化（增加动量、深度记忆、非线性递推）。 |
| 4 | **Mamba: Linear-Time Sequence Modeling with Selective State Spaces** (Gu & Dao, 2024) | 现代线性循环模型的代表。Titans 在几乎所有任务上超越 Mamba，理解 Mamba 有助于理解 Titans 的改进点。 |

### 值得一读

| # | 论文 | 推荐理由 |
|---|------|----------|
| 5 | **Memorizing Transformers** (Wu et al., 2022) | 外部 kNN 记忆的开创性工作。与 Titans 的"参数化记忆"形成对比——一个存原始 KV，一个学压缩表示。 |
| 6 | **Fast Weight Programmers** (Schmidhuber, 1992; Schlag et al., 2021) | 将线性层视为可写关联记忆的理论基础。Titans 的 delta-rule 更新直接源于此传统。 |
| 7 | **RULER: What's the Real Context Size of Your LLM?** (Hsieh et al., 2024) | Titans NIAH 实验使用的基准。揭示了"标称上下文长度"与"有效上下文长度"的差距。 |
| 8 | **BABILong: Long-context Benchmark** (Kuratov et al., 2024) | Titans 长上下文推理实验的基准。需要跨极长文档进行多跳推理，比 NIAH 更难。 |
