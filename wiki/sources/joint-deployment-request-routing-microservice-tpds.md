# Joint Deployment and Request Routing for Microservice Call Graphs in Data Centers

## 一句话总结

基于开放 Jackson 排队网络和细粒度容器编排，提出 GMDA-RMPR 两阶段启发式算法，联合优化数据中心中微服务调用图的部署与请求路由。

## 关键要点

- **核心问题**：微服务部署与请求路由相互耦合，传统方案将其视为独立问题只做局部优化，忽视了数据依赖和实例级部署
- **建模方法**：将联合微服务部署与请求路由问题（JMDRP）建模为混合整数非线性规划（MINLP），使用开放 Jackson 排队网络刻画排队延迟、处理延迟和通信延迟
- **GMDA 算法**（第一阶段）：贪心微服务部署算法，基于因子式资源分割和微服务级调用依赖，实现细粒度容器部署
- **RMPR 算法**（第二阶段）：请求匹配与分区映射算法，以概率方式将请求流分割并路由到微服务实例
- **实验效果**：平均部署成本降低 27.4%，端到端响应延迟降低 15.1%，请求成功率提升 5.47%
