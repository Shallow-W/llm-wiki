# Titans

## 定义

Titans 是由 Google Research 提出的一种**序列模型架构家族**，其核心创新是将**神经长期记忆模块**（Neural Long-term Memory）与**短期注意力**结合，实现测试时持续学习记忆。Titans 包含三种变体：MAC（Memory as a Context）、MAG（Memory as a Gate）和 MAL（Memory as a Layer）。

## 核心思想

Titans 的设计深受人类记忆系统启发：

> 人类记忆不是单一过程，而是由多个独立运作的子系统组成——短期记忆、工作记忆、长期记忆。一个有效的学习范式需要 distinct yet interconnected 的模块。

在这一框架下：
- **注意力（有限上下文）** = 短期记忆：精确建模依赖，但容量受限
- **神经记忆（深度 MLP）** = 长期记忆：持续学习，将历史抽象存储在参数中
- **持久记忆（可学习参数）** = 元记忆/任务知识：数据独立，编码任务本身的知识

## 关键属性

### 神经长期记忆模块 (LMM)

#### 1. 惊讶驱动的学习

记忆更新由"惊讶"驱动——输入与预期的差距越大，越值得记忆：

$$M_t = M_{t-1} + S_t$$
$$S_t = \underbrace{\eta_t S_{t-1}}_{\text{Past Surprise}} - \underbrace{\theta_t \nabla \ell(M_{t-1}; x_t)}_{\text{Momentary Surprise}}$$

- **Past Surprise**：动量项累积历史惊讶流，避免连续惊讶后的信息遗漏
- **Momentary Surprise**：当前梯度衡量输入的"意外程度"

#### 2. 自适应遗忘

$$M_t = (1 - \alpha_t) M_{t-1} + S_t$$

$\alpha_t \in [0,1]$ 数据依赖地控制遗忘程度：
- $\alpha_t \to 0$：保留旧记忆
- $\alpha_t \to 1$：清空记忆（上下文切换时）

#### 3. 深度记忆

使用 $L_M \geq 1$ 层 MLP 作为记忆模块，而非简单的向量/矩阵：

| 记忆深度 | 表达能力 | 长序列鲁棒性 | 训练速度 |
|----------|----------|-------------|----------|
| $L_M = 1$（线性） | 线性回归 | 较弱 | 最快 |
| $L_M = 2$ | MLP（严格强于线性） | 较好 | 较快 |
| $L_M = 3, 4$ | 更深抽象 | 最好 | 较慢 |

#### 4. 关联记忆目标

学习键值关联：

$$\ell(M_{t-1}; x_t) = \frac{1}{2} \|M_{t-1}(k_t) - v_t\|_2^2$$

在元学习框架中：内循环优化 $M$ 的权重，外循环优化 $W_K, W_V$ 和架构其他参数。

### 三种架构变体

#### MAC (Memory as a Context)

将长期记忆检索结果作为注意力的额外上下文：

```
输入分段 → 查询记忆 → [持久记忆 || 记忆检索结果 || 当前段] → 注意力 → 更新记忆
```

**特点**：
- 注意力自主决定是否需要长期记忆
- 注意力帮助筛选应记忆的信息
- 测试时：持久记忆固定、注意力上下文学习、长期记忆持续学习
- **长上下文表现最佳**

#### MAG (Memory as a Gate)

滑动窗口注意力（短期记忆）与神经记忆（长期记忆）通过门控融合：

```
输入 → [短期记忆分支 (SWA)] ──┐
       [长期记忆分支 (LMM)] ──┼──→ 门控融合 → 输出
       [持久记忆] ────────────┘
```

**特点**：
- 不分割序列，流式处理
- 两种记忆通过可学习门控动态权衡
- 效率与效果的良好平衡

#### MAL (Memory as a Layer)

传统堆叠方式：记忆层在前，注意力层在后：

```
输入 → 神经记忆层 → 滑动窗口注意力 → 输出
```

**特点**：
- 最传统的设计，与 H3、Samba 等类似
- 无法利用两种模块的互补数据处理
- 性能弱于 MAC/MAG

### 持久记忆 (Persistent Memory)

可学习但数据独立的参数 $P = [p_1, ..., p_{N_p}]$，拼接在序列开头：

$$x_{\text{new}} = [p_1, ..., p_{N_p}] \| x$$

三重作用：
1. 存储任务知识（如何完成任务的抽象）
2. 类似 FFN 中数据独立的"注意力权重"
3. 缓解因果注意力对初始 token 的偏置

## 并行化训练

神经记忆的训练等价于带动量+权重衰减的在线梯度下降。通过以下技巧实现并行化：

1. **分块 mini-batch**：将序列分为块，利用 matmul 重写更新
2. **关联扫描**：动量递推 $S_t = \eta_t S_{t-1} - \theta_t u_t$ 用并行关联扫描在 $O(\log N)$ 深度完成
3. **块级常数参数**：将门控参数设为块内常数，可用全局卷积加速

## 理论性质

**Theorem**：Titans 能解决 TC$^0$ 之外的问题，而 Transformer、对角线性循环模型和 DeltaNet 均限于 TC$^0$。这意味着 Titans 在状态跟踪任务中理论上更具表达能力。

## 性能表现

| 模型 | Perplexity (Wiki) | 常识推理 Avg. |
|------|-------------------|---------------|
| Transformer++ | 25.21 | 48.69 |
| Gated DeltaNet-H2 | 19.88 | 51.49 |
| **Titans (MAC)** | **19.93** | **52.51** |
| **Titans (MAG)** | **18.61** | **52.50** |

| 模型 | S-NIAH-PK | S-NIAH-N | S-NIAH-W |
|------|-----------|----------|----------|
| Mamba2 | 5.4 | 0.0 | 0.0 |
| DeltaNet | 71.4 | 5.4 | 0.0 |

### BABILong（跨文档推理）

Titans (MAC) 超越 GPT-4、Qwen2.5-72B、Llama3.1-70B 等超大模型。
